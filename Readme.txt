1. Packages to be installed are below. Use below command to install all the packages. ttkbootstrap is for GUI, matplotlib is for the plots and networkx to create a random graph on which our algorithms would work.

pip install ttkbootstrap matplotlib networkx

2. shortestPathAlgorithmsGUI.py is the main application. Please run this to see the GUI and test the logic.
3. shortestPathAlgorithms.py is the backend code which has all the main algorithms implemented
